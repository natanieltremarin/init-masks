How to use

Masks used in sending get forms.

To get the small lib working correctly, you must have an updated version of the browser, and internet explorer is not supported.

Types of masks:
  • Pt-br
    • Cnpj > i-mask="cnpj"
    • Cpf > i-mask="cpf"
    • Cartão de Crédito > i-mask="credit-card"
    • Data > i-mask="data"
    • Fone > i-mask="fone"
  • En
  • Es
Mandatory attributes
  • name="" (Unique identifier of the field per page).
  • i-mask="" (The identifier of the desired mask).
  • type="text" (For correct operation the type attribute must be text).

Recommended additional attributes
  • minlength="" (Please specify if you want to validate the minimum number of characters, but count the spaces generated that will be removed when sending the content).
  • required (If it is a required field).
